# Valorian — Figma Website Builder Prompt

## Project Overview

Redesign the website for **Valorian** (formerly Valorian Circle) — a curated European network for leaders in technology, entrepreneurship, investment, and public policy. The site should feel like a mission-driven institution, not a membership club. Reference tone and emotional register: YPO (ypo.org), but with a distinctly European identity. Prestige without excess. Clarity with depth.

---

## Brand Identity

**Name:** Valorian (remove "Circle" from all instances)

**CTA label:** Use "Request Invitation" consistently throughout — never "Apply" or "Apply for Membership"

**Color palette:**

- Primary: European Navy #0D1F3C
- Accent: Hunter Green #355E3B — used sparingly as a tertiary accent color
- Background: Off-white #F9F8F5
- Text: Near-black #1A1A1A

The navy + green combination is intentional. Use green for criteria headings, select accents, and decorative elements. Never as a background fill.

**Typography:**

- Headlines: Cormorant Garamond — retain the current site's serif italic + upright editorial combination
- Body: Inter
- Keep italic emphasis style throughout (e.g., *A Vision for* **Europe.**)

**Copy rules:**

- No em dashes in running text — resolve through rephrasing or a colon
- Hyphens retained for compound modifiers: 100-day, co-found, invitation-only
- No abstract mission language — every sentence names a concrete action or outcome
- Tone: institutional conviction, not membership club warmth

## Overall Design Direction

The current site has strong bones: clean serif typography, generous whitespace, restrained layout. This rebrand is a color and mission shift, not a structural overhaul.

Key changes:
- Dark green replaced by European Navy throughout
- Hero copy shifts from transactional to mission-driven
- Product architecture: single track becomes two named tracks (Circles, Fellowships)
- Membership criteria displayed as a visual grid, not a text block
- Emotional register: from private members club to European leadership institution

Do not over-design. The current minimalism is an asset. The navy and gold palette, updated copy, and new Fellowships track are sufficient to signal a meaningful evolution while preserving continuity for existing members.
